Key facts: Vietnamese-first (`lang="vi"`), corporate aesthetic v3 "Corporate Blue" for an older, skeptical B2B audience. Brand blue #368FFF (base); buttons/text use blue-600 #2374D9+ for AA; hero/CTA bands blue-900 #0D2F5E; slate neutrals; white footer (KiotViet-style: company info + map + 4 link columns). NO yellow/amber CTAs — all actions are brand blue. Playfair Display serif for display/h1/h2 (Vietnamese subset), IBM Plex Sans body, IBM Plex Mono for money/sizes/scores (VN format "45.000.000đ/tháng"). Sharp 1–6px radii (pills excepted), hairline borders, restrained motion (150/250/400ms), AA contrast, status/AI-score never color-only. Site nav is 5 fixed items: Trang chủ · Bản đồ · Dịch vụ · Tin tức · Liên hệ (separate pages, shared header/footer). No emoji, no purple gradients, no em-dash in visible copy, no absolute marketing claims.

If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.
